# Ready Room Icon Concept Prompts

These are the exact prompts used for the concept-generation dry runs.

## Glass Dashboard Tile

Use case: logo-brand  
Asset type: macOS Dock and Finder app icon  
Primary request: Ready Room app icon concept called "Glass Dashboard Tile" showing a symbol-only command-center console with stacked glass dashboard cards and one briefing sheet panel  
Style/medium: premium macOS glass icon, polished, restrained, native-feeling  
Composition/framing: centered rounded-square app icon silhouette, one central console shape, layered panels, thin left accent rail  
Lighting/mood: calm, premium, soft pearl highlights, subtle depth  
Color palette: deep slate and navy glass with restrained accents in John blue `#3478F6`, Amy green `#39A96B`, Ellie lavender `#B58AF7`, and Mia light blue `#7ECFFF`  
Constraints: symbol only, no text, no RR monogram, no people, no house outline, no shield, no literal calendar date, no big envelope, no loud rainbow background, no clutter  
Avoid: cartoon feel, loud gradients, neon glow, busy interface details, generic mail icon, generic family logo
