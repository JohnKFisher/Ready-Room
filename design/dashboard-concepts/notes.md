# Concept Notes

## Spread Check

- Conservative set: keeps the familiar split-shell mental model while testing polish, atmosphere, and visual calm.
- Medium set: tests whether Ready Room should move toward an operations board, a bento canvas, or clearer day-banded scanning.
- Big swings: tests whether the app’s eventual identity wants to feel cinematic, editorial, or kiosk-like.

## Quick Read

- Most implementation-friendly shortlist: `Morning Ledger`, `Quiet Columns`, `Signal Board`
- Best “interesting but still plausible” shortlist: `Glass Rail`, `Bento Pulse`, `Dayline Focus`
- Best identity exploration shortlist: `Command Theater`, `Editorial Desk`, `Beacon Wall`

## What To Look For In Review

- Which concepts make `Today` easiest to scan in under five seconds?
- Which concepts keep secondary modules readable without stealing focus from the timeline?
- Which concepts feel strong enough to become the final design language instead of just a temporary facelift?
- Which concepts look best from a desk versus across the room?
- Which concepts imply kiosk-only behavior or a separate wall-display mode rather than a single dashboard for every use?

## Suggested Down-Select Path

1. Pick one conservative favorite as the “safe implementation” anchor.
2. Pick one medium favorite as the “worth prototyping in-app” candidate.
3. Pick one big swing as the “identity stretch goal.”
4. If we move to implementation later, build the winner first in a separate sandbox screen instead of editing the production dashboard directly.
